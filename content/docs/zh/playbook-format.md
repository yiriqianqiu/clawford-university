# 学习剧本格式

学习剧本是端到端的学习指南，采用人机协作模式：人类负责决策，Agent 负责执行。

## 结构

每个剧本是一个 MDX 文件，存放在 `playbooks/<locale>/playbook-<name>.mdx`。

## 章节说明

### 1. 概览

- 标题和一句话描述
- 目标受众
- 预计时间
- 难度级别（初级 / 中级 / 高级）

### 2. 痛点

这个剧本解决什么问题？用户正在为什么发愁？

### 3. 学习逻辑

基于证据的推理，说明为什么这套方法有效。

### 4. 人机分工

清晰的职责划分：
- **人类负责** — 决策、目标设定、质量判断
- **Agent 负责** — 调研、起草、分析、执行

### 5. 步骤（3-5 步）

每个步骤包含：
- 明确的目标
- 人类的操作
- Agent 的操作
- 预期产出

### 6. 技能包

需要安装的技能包：
```
clawford install @clawford/google-search @clawford/summarizer
```

### 7. 预期产出

完成整个剧本后，用户应该产出什么成果。

## 分类

- **通用** — 学习、写作、编程、研究
- **加密** — 交易、DeFi、KOL 运营、代币发射

## 多语言支持

剧本提供英文版（`playbooks/en/`）和中文版（`playbooks/zh/`）。系统会根据用户的语言设置自动选择对应版本。
