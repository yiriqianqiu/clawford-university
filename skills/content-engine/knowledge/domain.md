# Content Engine — Domain Knowledge

## Platform Rules

### X (Twitter)
- 280 chars per tweet, threads for depth
- Strong hooks in first line — stop the scroll
- One idea per tweet, opinion > information
- Use line breaks for readability
- Engagement bait is not engagement — be genuine

### LinkedIn
- Professional tone, but human
- Story format works best (situation → action → result)
- First 3 lines must hook (before "see more")
- 1,300-1,500 chars sweet spot

### YouTube
- Title: curiosity gap or clear value proposition
- Script: hook (15s) → context (30s) → meat → CTA
- Description: keywords + timestamps + links

### Newsletter
- Subject line = open rate. Test 3 variants.
- One topic per issue. Deep > wide.
- End with a question or action.

## Anti-Patterns

- "In today's fast-paced world..." — instant skip
- Posting the same text everywhere — platform mismatch
- All value, no personality — boring
- All personality, no value — empty
- Posting without a hook — invisible

## Repurposing Flow

```
Anchor Asset (blog post, video, research)
  → Extract 3-7 atomic ideas
  → For each idea:
    → X: punchy take + thread if deep
    → LinkedIn: story or insight format
    → YouTube: short or long script
    → Newsletter: deep dive section
```
