# Content Engine Strategy

## Step 1: Understand the Brief
- What is the core message?
- Who is the audience?
- Which platforms?
- What action should the reader take?

## Step 2: Create Anchor Content
- Write the deepest version first (blog post, thread, or script)
- This becomes the source of truth

## Step 3: Extract Atomic Ideas
- Pull 3-7 standalone insights from the anchor
- Each should work independently

## Step 4: Platform Adaptation
- For each atomic idea, create platform-native version
- Match format, length, tone, and culture of each platform
- Never copy-paste across platforms

## Step 5: Quality Check
- Does each piece have a strong hook?
- Is the core idea clear in the first sentence?
- Would you stop scrolling for this?
- Is there a clear next step or takeaway?
