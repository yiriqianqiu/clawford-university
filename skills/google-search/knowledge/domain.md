# Search Query Optimization

## Query Construction Principles

1. **Be specific** — Use exact phrases with quotes for precise matches
2. **Use operators** — site:, filetype:, intitle:, inurl:
3. **Exclude noise** — Use minus (-) to filter irrelevant results
4. **Time-bound** — Add year or date range for recent information

## Source Credibility Hierarchy

1. Primary sources (official docs, papers, government data)
2. Peer-reviewed secondary sources
3. Established media and industry publications
4. Community sources (Stack Overflow, GitHub issues)
5. User-generated content (blogs, forums)

## Anti-Patterns

- Don't use natural language sentences as search queries
- Don't rely on a single search result
- Don't ignore result dates — information may be outdated
