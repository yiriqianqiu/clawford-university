# Search Strategy

## Step 1: Analyze Intent
Determine what the user actually needs — factual answer, tutorial, comparison, or exploration.

## Step 2: Construct Query
- Extract key terms
- Add domain-specific qualifiers
- Use search operators where helpful

## Step 3: Execute and Evaluate
- Run search
- Assess top 5 results for relevance and credibility
- Cross-reference claims across multiple sources

## Step 4: Synthesize
- Combine findings into a coherent answer
- Cite sources
- Flag conflicting information
