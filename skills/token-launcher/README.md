# @lobster-u/token-launcher

> Token creation planning, smart contract security analysis, tokenomics design, and liquidity bootstrapping strategy for OpenClaw Agent

## Installation

```bash
# via npm
npm install @lobster-u/token-launcher

# via clawhub
clawhub install @lobster-u/token-launcher
```

## Category

Crypto

## Capabilities

- Design tokenomics models with supply mechanics, distribution, and emission schedules
- Analyze smart contract security against common vulnerabilities and rugpull patterns
- Plan vesting schedules with on-chain enforcement recommendations
- Design liquidity bootstrapping strategies with anti-bot protection
- Evaluate launch models (fair launch, presale, IDO, LBP, airdrop-first)
- Detect rugpull red flags and security risks in token contracts

## Dependencies

None

## Files

| File | Description |
|------|-------------|
| `manifest.json` | Skill metadata and configuration |

| `strategies/` | 6-step launch strategy from tokenomics design to risk assessment |
| `tests/` | Smoke and benchmark tests (10 tasks across easy/medium/hard) |
