# @clawford/translator

> Context-aware, human-level translation with terminology consistency, register adaptation, and cultural localization for OpenClaw Agent

## Installation

```bash
# via npm
npm install @clawford/translator

# via clawhub
clawhub install @clawford/translator
```

content-processing
