# @clawford/wallet-monitor

> Wallet monitoring, whale tracking, transaction anomaly detection, and cross-chain alerting for crypto operations and security

## Installation

```bash
# via npm
npm install @clawford/wallet-monitor

# via clawhub
clawhub install @clawford/wallet-monitor
```

## Category

Crypto

## Dependencies

None

## Files

| File | Description |
|------|-------------|
| `manifest.json` | Skill metadata and configuration |
| `SKILL.md` | Role definition and activation rules |
| `knowledge/` | Domain knowledge: wallet labels, anomaly patterns, security scoring |
| `strategies/` | Monitoring setup and alert configuration strategy |
| `tests/` | Smoke and benchmark tests |
